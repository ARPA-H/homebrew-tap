# ARPA-H Homebrew Tap

This is the official [Homebrew](https://brew.sh) tap for ARPA-H tools. It lets macOS users install ARPA-H-built command-line tools with a single `brew install` command.

## What Is a Homebrew Tap?

[Homebrew](https://brew.sh) is the standard package manager for macOS. Its default "core" registry only includes widely-used open-source tools. A **tap** is a third-party package repository — any GitHub repo named `<org>/homebrew-<name>` — that Homebrew can pull in to extend its package catalog.

```
brew tap ARPA-H/tap
# Homebrew now knows about all formulas in this repo
```

Once tapped, packages from this repo can be installed, upgraded, and managed just like any other Homebrew package.

### How It Works

```
┌─────────────────────────────┐
│   Source repo (e.g. jira-   │
│   copilot) cuts a release   │
│   and uploads binaries      │
└──────────────┬──────────────┘
               │ GitHub Actions triggers
               ▼
┌─────────────────────────────┐
│   homebrew-tap              │
│   Formula/*.rb auto-updated │
│   (version + sha256 bumped) │
└──────────────┬──────────────┘
               │ user runs brew install
               ▼
┌─────────────────────────────┐
│   Homebrew downloads the    │
│   binary, verifies sha256,  │
│   and installs it           │
└─────────────────────────────┘
```

Each package is defined by a **formula** — a small Ruby file in `Formula/` that tells Homebrew where to download the binary, how to verify its integrity, and how to install it. Formula files in this repo are **auto-generated by GitHub Actions** and should not be edited by hand.

## Available Tools

| Formula | Command | Description |
|---------|---------|-------------|
| [`jcc`](Formula/jcc.rb) | `jcc` | **Jira Cloud Copilot** — terminal UI for Jira workflows: standup, tickets, dates, and activity |

## Installation

### 1. Add the tap (one-time setup)

```sh
brew tap ARPA-H/tap
```

### 2. Install a tool

```sh
brew install jcc
```

Or, without adding the tap first:

```sh
brew install ARPA-H/tap/jcc
```

### Updating

```sh
brew update && brew upgrade jcc
```

## Repository Structure

```
homebrew-tap/
└── Formula/
    └── jcc.rb        # Auto-generated formula for jcc (Jira Cloud Copilot)
```

Formula files are updated automatically when a new release is published in the source repository. Each formula pins an exact version and SHA-256 checksum so Homebrew can verify the download before installation.

## For Maintainers: Adding or Updating a Formula

Formula files are managed by GitHub Actions in the source repositories. To add a new tool to this tap:

1. In the tool's source repo, set up a release workflow that:
   - Builds and uploads a `.tar.gz` binary for each target architecture
   - Computes the SHA-256 of each archive
   - Opens a PR (or pushes directly) to this repo updating `Formula/<toolname>.rb`
2. The formula should follow the pattern in [`Formula/jcc.rb`](Formula/jcc.rb).

**Do not edit formula files by hand** — they will be overwritten on the next automated build.
